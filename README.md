# My Projects

### 1. SHPE Mobile App
A mobile application developed for the Society of Hispanic Professional Engineers (SHPE) UCF chapter to help members access events, announcements, and chapter information. 
Contributed to front-end development and API integration, including building layouts and connecting APIs.
- Tech used: Dart, Flutter, Firebase
- Features: User account creation and login, view upcoming SHPE events, Instagram-style feed displaying chapter activities, Dynamic UI built with Flutter, API integration for retrieving event and user data

### 2. Wear of Tear
An application that evaluates a person’s outfit and determines whether it meets specific style or condition guidelines.
- Tech used: Python
- Features: User login and authentication, identification of outfits, AI-based review and feedback on outfits

### 3. Team Contact Manager 
A full-stack web application that allows users to securely manage personal contacts, including creating, editing, deleting, and searching contacts.
- Tech used: Linux, Apache, MySQL, PHP, HTML, CSS, JavaScript
- Features user authentication and deployment on a DigitalOcean server

### 4. Brulee Web App
A web application built for a business to create a responsive webpage for showcasing services and business information, with dynamic functionality and backend support.
- Tech used: JavaScript, HTML, Python, CSS
- Features: Responsive business webpage layout, Interactive UI using JavaScript, backend functionality using Python, display of business services and information, styled user interface with CSS

### 5. CNN Image Classifier
A deep learning project that trains a three-layer Convolutional Neural Network (CNN) to classify images from datasets such as CIFAR-10 or MNIST. The project explores data augmentation, normalization, and model evaluation.
- Tech used: Python, PyTorch
- Features: Image classification using a CNN, data augmentation and preprocessing, training and evaluation with a confusion matrix
